# Cosy Mood Journal

A minimal and cosy web app to track your mood and write short reflections. 

This project is part of my journey as a first-year Computer Science student, building small projects to gain experience for a future year in industry.

## Features
- Mood Selection (happy, sad, angry, relaxed, anxious)
- Reflection textarea for journaling

## Built With
- HTML
- CSS *(in progress)*
- JavaScript *(in progress)*

## Planned Improvements

- Improve styling
